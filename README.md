--- REQUIRED DEPENDENCIES ---
1. streamlit  - Web framework for the dashboard
2. polars     - High-performance data processing
3. pandas     - Required for Plotly-to-Streamlit integration
4. plotly     - Interactive data visualizations
5. dacite     - Complex JSON to Dataclass mapping

--- INSTALLATION ---
Run the following command in your terminal to install everything:

pip install streamlit polars pandas plotly dacite

--- HOW TO RUN ---
1. Navigate to the project root directory (where app.py is located).
2. Execute the following command:

streamlit run app.py
